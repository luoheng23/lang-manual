# lang-manual
This is a project to show difference or similarity of lots of languages. You can find
useful functions implementing in different languages.

# How to compile these programs
```bash
gcc hello.c && ./a.out
g++ hello.cpp && ./a.out
go run hello.go
ghc hello.hs && ./hello
java hello.java
perl hello.pl
python hello.py
rustc hello.rs && ./hello
swift hello.swift
```
